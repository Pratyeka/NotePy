### 封装 ###
- 将多个使用逗号分割的数据组合起来
- 本质上返回的是一个元祖类型

### 解构 ###
- 将线性解构的元素解开，并顺序的复制给其他变量
    - 非线性的数据结构也可以被解构，顺序无法保证
    - 字典拿到的是key值。不是value值
- 要保证封装的元素个数与赋值的变量个数完全一致
```Python
    a, b = 3, 56
    print(a, b)
    a, b = (3, 56)
    print(a, b)
    a, b = {3, 4}     # 解构非线性的数据类型
    print(a, b)
    a,b = {'a':2, 'b':54}    # 解构非线性的数据类型，得到键名
    print(a, b)
```

- 可以使用`*name`形式接受
    - `*name` 形式不能单独使用
    - `*name` 形式只能出现一次
    - `*name` 形式接受的元素，构成一个列表
```Python
    *a, b = (1,2,3)
    print(a,b)
    a, *b = (1,2,3)
    print(a,b)

    a, *c, b = (1,2,3,4)
    print(a,b)
    a, *_, b = (1,2,3,4)
    print(a,b)
```

### 丢弃变量 ###
- `_`：是Pythoner约定好的一个可丢弃变量，表示该变量不关心

*要注意丢弃变量使用时的作用范围，防止与库中定义的_冲突*
